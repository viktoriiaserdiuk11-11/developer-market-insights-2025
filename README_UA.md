# Developer Market Insights 2025

## Про проєкт

Цей проєкт створений як практичний дослідницький аналіз даних, проєкт на основі датасету Stack Overflow Developer Survey 2025.

Метою було не просто побудувати графіки, а пройти повний  аналітичний процес:
- завантаження та перевірка сирих даних опитування
- перевірка пропущених значень і дублікатів
- очищення неповних записів
- дослідження розподілів та трендів
- побудова агрегованих метрик
- створення візуальних інсайтів у Tableau Public
- представлення результатів у структурованому дашборді

Аналіз фокусується на зарплатах розробників, форматах роботи, популярності мов програмування, розподілі індустрій та використанні Python серед різних вікових груп.

---

## Підготовка даних

Перед аналізом датасет був оброблений та очищений:
- перевірено дублікати респондентів
- проаналізовано пропущені значення
- видалено неповні записи в окремих частинах аналізу
- підготовлено поля для агрегації та візуалізації

Проєкт використовує exploratory analysis methods (методи дослідницького аналізу) та фокусується на виявленні закономірностей у відповідях респондентів.

## Досліджені питання

- У яких країнах найвищі медіанні зарплати розробників?
- Які мови програмування використовуються найчастіше?
- Які формати роботи зараз найпопулярніші?
- Які індустрії наймають найбільше розробників?
- У яких вікових групах найвищий рівень використання Python?

---

## Основні інсайти

- Remote (віддалений) та hybrid (гібридний) формати роботи залишаються дуже популярними серед розробників.
- Зарплати розробників суттєво відрізняються між країнами.
- Python частіше використовувався серед молодших вікових груп у датасеті.
- Software Development (розробка програмного забезпечення) показала одне з найсильніших поєднань популярності remote-роботи та вищих рівнів зарплат.

---

## Обмеження аналізу

- Аналіз базується на відповідях респондентів опитування та може не повністю відображати глобальний ринок розробників.
- Частина інсайтів є descriptive (описовими) і створена для практики exploratory analysis (дослідницького аналізу).
- Значення зарплат і робочих вподобань залежать від self-reported data (даних, які користувачі вказували самостійно).
- Дашборд варто інтерпретувати як exploratory analytics (дослідницьку аналітику), а не як бізнес-прогнозування.
